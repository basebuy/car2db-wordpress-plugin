== Description ==

Car Make, Model, Serie selection widget with ajax autocomplete from the database. Uses Car2DB data structure: 
information bank for the online-store or classified website. The performance of engine, transmission, size and weight.

Demo database for this widget can be downloaded here: https://car2db.com/#interactive
Full database not included with this widget and can be purchased here: https://car2db.com/#buy

== Installation ==

1. Upload `car2db` plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place Car2DB widget to WordPress theme
